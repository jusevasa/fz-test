# How to start aplication

1. install dependencies
```
yarn install
```
2. copy .env.example and remove the text .example for generate the variables

3. run dev server
```
yarn dev
```

# How to run test

1. run test all test
```
yarn test
```

2. run coverage aplication
```
yarn coverage
```

3. run test E2E
```
yarn cypress
```
